# ttd

A interactive CLI Todo App in Rust

## Quick Start
```console
$ cargo run <file_path>
```

## Example
```console
$ cargo run SAMPLE.txt
```

## Controls

|Keys|Description|
|---|---|
|<kbd>w</kbd>, <kbd>z</kbd>|Move up|
|<kbd>s</kbd>|Move down|
|<kbd>Return</kbd>|Toggle selected element's state|
|<kbd>Tab</kbd>|Switch between tabs|
|<kbd>q</kbd>|Quit|
